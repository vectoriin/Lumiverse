# Quick Replies

Quick replies are pre-written message templates you can insert into the chat with a single click. Useful for common responses, recurring actions, or frequently used OOC instructions.

---

## Setting Up Quick Replies

1. Open **Settings > Quick Replies**
2. Click **New Set** to create a collection
3. Give the set a name (e.g., "Combat Actions," "Common Responses," "OOC Commands")
4. Add replies — each with a **label** (what you see in the menu) and a **message** (what gets inserted)
5. Toggle the set **on** to make it available in the chat input

---

## Using Quick Replies in Chat

1. Click the **Quick Reply** icon in the input area action bar
2. Browse your enabled sets
3. Click any reply label
4. The message is inserted into the input field — edit it if needed, then send

Quick replies don't send automatically. They fill the input area so you can review or modify before sending.

---

## Organizing

- **Sets** group related replies together
- **Color tags** help visually distinguish sets
- **Enable/disable** entire sets to keep the menu clean
- Click **Manage in settings** from the popover to jump to the configuration page
