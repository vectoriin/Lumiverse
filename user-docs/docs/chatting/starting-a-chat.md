# Starting a Chat

Every conversation in Lumiverse begins with selecting a character and creating a chat.

---

## Creating a New Chat

There are several ways to start a chat:

### From the Character Browser

1. Open the **Character Browser** panel
2. Click on a character card
3. A new chat is created automatically with the character's first message

### From the Landing Page

1. Click on a character's avatar on the landing page
2. Choose **New Chat** (or select an existing chat to continue)

### From an Existing Chat

You can switch characters or start fresh at any time from the chat view.

---

## Greeting Selection

If a character has **alternate greetings**, you'll be prompted to choose one when starting a new chat. Each greeting sets a different opening scenario.

Characters can have as many alternate greetings as the creator wants — each one is a different "hook" for the conversation.

---

## Chat Names

Chats are automatically named based on the character, but you can rename them for easier identification. Useful when you have multiple chats with the same character for different scenarios.

---

## Resuming Chats

Your chats are saved automatically. To resume:

- **Landing Page** — Shows your most recent chats with previews
- **Manage Chats** modal — Search and filter through all your chats
- **Character Browser** — Click a character, then choose from their existing chats

---

## What Gets Saved

Every chat stores:

- All messages (yours and the character's)
- All swipe variants for each message
- Author's Note content and settings
- Alternate field selections
- Active avatar selection
- Active expression state
- Macro variables (local to the chat)
- World Info activation state (sticky/cooldown counters)
- Loom summary data
