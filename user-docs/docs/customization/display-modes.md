# Display Modes

Lumiverse offers two visual modes for how messages appear in the chat. Choose the one that fits your aesthetic preference and reading style.

---

## Available Modes

### Minimal

A clean, text-focused layout with minimal visual chrome. Messages are distinguished by subtle accent bars on the left (character) and right (user). Compact spacing makes it ideal for long conversations where you want to focus on the text.

### Bubble

A messaging-app style layout where each message appears in a rounded bubble. Character messages appear on the left with avatars; user messages appear on the right (configurable). Feels more conversational and visually distinct.

**Bubble-mode options:**

| Setting | Description |
|---------|-------------|
| **User Message Alignment** | Left or Right (default: Right) |
| **Show character art in bubble backgrounds** | Uses the character's avatar as a soft, dissolving background behind their bubbles. Turn off for a flatter, content-first look. |

---

## Chat Width

Both modes share the same width control. The chat content area can be constrained on wide monitors so messages don't stretch edge-to-edge.

| Width | Description |
|-------|-------------|
| **Full** | Messages use the full available width |
| **Comfortable** | Narrower content area for easier reading |
| **Compact** | Even narrower — good for very wide monitors |
| **Custom** | Pick an exact pixel width with a slider (500–2000 px) |

---

## Configuring

Open **Settings → Chat** to choose your display mode, bubble options, and width preference. Changes apply immediately to the active chat.
